이클립스에서 프로젝트를 생성하고 git으로 버전관리를 원하면  우측상단의 git 클릭하고 새로운 로컬저장소 만들기 생성함 경로는 자바 프로젝트 경로폴더까지
연결 끊고 싶으면 우클릭해서 team -> disconnect.. 다시 연결하고 싶으면 team -> 쉐어링 ㄱㄱ


간단하게 정리 github와 연결
- GitHub 레퍼지토리 생성
- 본인 컴퓨터에 복사하기 (" git clone https://github.com/ddahyeon/java_study.git ") 
- clone한 파일 안에서 프로젝트 생성 후 작업 (어떤 IDE를 사용하던지 상관없음)
- 작업 파일안에 .gitignore 파일을 만들고 위의 텍스트 복붙 -> 필요없는 파일은 올리지 않으려고 하는 것(해도되고 안해도 됨)
- 작업이 다 끝났으면 터미널에서 
    git add . (.은 파일 다 올리는건데 git은 애초에 수정된 파일만 commit하므로 .이걸로 해도 됨 -> 파일 지정하고 싶으면 git aad 파일명 )
    git commit -m "커밋메시지" 
    git push origin main 
해주면 자동으로 github에 올라감 

** 기존 파일에 수정 후 git push 할 때 오류날 수도 있는데 '덮어쓰기 방지' 뜨는거라서 git에 있는 내용을 먼저 가져와서 합치고 push 해줘야함 
git pull origin main --rebase
git push origin main
해주면 정상적으로 가능 

잘못올렸을때 git restore/reset 하면 됨
